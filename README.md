# DevSecOps Jenkins Pipeline Demo

This project demonstrates a simple DevSecOps CI/CD pipeline using Jenkins with security and quality checks.

## Tools Used
- Jenkins
- Git
- SonarQube
- Docker
- AWS (conceptual)
- Linux

## Pipeline Stages
- Checkout Code
- Build
- SonarQube Scan
- Docker Build
- Deploy (demo)

## Outcome
Shows how security and quality can be integrated early in SDLC (shift-left DevSecOps).
